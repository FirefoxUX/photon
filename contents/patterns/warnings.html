Warnings

Components using warning states:

  • Message bars
  • Notification bars
  • Input field
  • Safe browsing pages

Usage

Warnings provide short and relevant information that adds context when a user encounters a problem or issue. Whenever possible, provide the user with an action that helps them address the warning.

Placement

Warnings are typically placed close to the exception requiring user action.

Style

Icon

An exclamation mark enclosed inside a triangle indicates a warning. The icon is typically paired with warning text.

Color

Yellow is the dedicated color for warning states. Warning text should be highly legible, with noticeable contrast against its background color.
