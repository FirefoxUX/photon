Warnings

Components using warning states:

  • Message bars
  • Notification bars
  • Input field
  • Safe browsing pages

Usage

Warnings provide short and relevant information about a specific situation. If possible, give to the user an action that will help them address the warning.

Placement

Warnings are usually place close to the context.

Style

Icon

Warnings text come together with an exclamation mark enclosed within a triangle.

Color

Yellow is the dedicated color for warning states. Warning text should be legible, with noticeable contrast against its background color.

Background

Yellow 50 #ffe900

Content

Yellow 90 #3e2800

High Alert Warnings

In rare case of high alert, it's possible to use the Red as a color for indicate the warning.

Background

Red 50 #ff0039, or trasparent

Content

White #ffffff, or Red 60 #d70022
