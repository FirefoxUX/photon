Errors

Errors (or error condition) occur when an the software fails to complete an expected action. Every error becomes a point of friction for the user.

Usage

Error messages provide short and relevant information that adds context when a user encounters a problem or issue. Whenever possible, provide the user with an action that helps them address the error.

It's important to distinguished two different kind of error:

  • User input errors
  • System errors

User input error occur when Firefox doesn't understand user inputs.

System errors occurs when Firefox fails for some reasons most likely not related to the user.

Errors are usually place close to the context.

Style

Icon

The icon used for indicate errors is an exclamation mark enclosed within a triangle, usually white on a red background, or red.

Colors

Red is the dedicated color for errors states. Warning text should be legible, with noticeable contrast against its background color.

Background

Red 50 #ff0039, or trasparent

Content

White #ffffff, or Red 60 #d70022
