Link

Link is text that users can click or tap to access websites or subsequent pages.

Usage

Use text links to guide users to a new page or to anchor users to a section within a page. For an action that results in an immediate change, use a button instead. See buttons

Types

Internal link

An internal link directs users to a page inside the Mozilla properties (e.g., SUMO, Pocket) or anchors users to a specific section on the same page.

External link

An external link directs users to a page not produced by or affiliated with Mozilla.

An Open In New icon should be paired with the external link.
